# Contributing Guidelines (Internal Only)

This project does NOT accept public contributions.

## Internal Contribution Rules
- Follow branch naming conventions: feature/*, fix/*, docs/*
- All work must go through Pull Requests.
- Lint, type-check, and test before submitting.

## Commit Messages
Use Conventional Commits:
- feat: new feature
- fix: bug fix
- docs: documentation updates
- chore: tooling/config
